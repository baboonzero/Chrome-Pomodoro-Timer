Pomodoro Timer Test Page

This page is designed to test the Pomodoro Timer extension overlay functionality. The overlay should appear in the top-right corner when the timer is running.

To test:

  1. Make sure the Pomodoro Timer extension is installed and enabled
  2. Click the extension icon in your browser toolbar
  3. Start the timer from the popup
  4. The overlay should appear on this page

Debugging:

  • Open Chrome DevTools (F12)
  • Check the Console tab for debug messages
  • Look for messages starting with "[Pomodoro Overlay]" and "[Pomodoro Background]"
  • If the overlay doesn't appear, check for error messages
